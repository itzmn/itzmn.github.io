交换类

快速排序

思路：

  1. 从待排序的数组中选择一个数据为基准数，通常选择数组第一位
  2. 把小于基准数的放在基准数左边，把大于基准数的放在右边，中间放基准数，然后基准数左右两边数组分别进行循环此步骤。

具体步骤：

快速排序，在数组中选定一个基准数，一般选择数组第一位，分两个方向查询，一个从左至右（索引为0），一个从右至左（索引为数组长度），先从右向左查询比基准数小的数（直到右边的索引和左边的索引相遇），查询到之后将此数赋值给基准数（记住此数，后面要用），然后开始从左至右查询比基准数大的数（知道左边的索引与右边的索引相遇），查询到之后将这个数赋值给刚刚的记住的数的位置，直到两索引相遇，然后将基准数赋值给中间值。这就完成了一次快排，然后从这个位置将数组分为左右两部分，分别进行快速排序。

冒泡排序

思路：

  1. 从数组第一个数据向后，两两比较，如果前面一个数据大，那么交换位置，直到最后，将最大值挪到最后一个位置完成一次循环
  2. 后面循环还是从第一个数据向后比较，两两比较，但是不需要比较到数组最后一位，比较到数组长度减去循环次数

具体步骤：

从第一个数据开始向后比较，如果前一个大就交换位置，如果后一个大就不交换,这是第一次，将最大值转移到了最后一位，然后开始第二轮，从第一个数据开始比较,但是至于要比较到倒数第二个数据，因为最后一个已经是最大的了

直接插入排序

思路：

  1. 将未排序数组插入有序数组，默认无序数组的第一位为有序数组。

具体步骤：

将无序数组的第一位认为是有序数组，然后从第二位开始将数据插入有序数组，如果第二位大于前面一位，则前两位数据是有序的，索引增加，将第三位数据与前面一个数据比较，如果第三位小于前面一位，则将前面一位数据赋值给第三位，然后索引减少，直到找到合适的位置，将数据插入

希尔排序

希尔排序是对直接插入排序的增强版，直接插入需要循环n-1次，每次都需要与前面的数据循环比较，

而希尔排序采用了增量的方式对直接插入排序改进

思路：

  1. 首先设置增量为数组的一半，然后根据增量进行分组
  2. 在每个组内进行直接插入排序
  3. 然后增量减少，循环以上两步，直至增量小于1

选择排序

思路：

  1. 假设最大值得索引为数组第一位，然后循环比较数组后面的值与最大索引的值，如果后面的值比较大记录下最大值索引，然后继续比较后面的值与最大值索引的值，循环到最后，则最大值索引处的值就是数组的最大值，将该数据与数组最后一位交换，第一次比较完成
  2. 假设最大值索引为第一位，然后重复上一步，不过只需要循环比较到倒数第二位，因为倒数第一位已经是最大值了，然后将这次的最大值和倒数第二位交换，
  3. 循环以上步骤即可

堆排序

思路

  1. 首先需要将数组建成一个堆，（大根堆）
  2. 将该数组的第一个数据与数组最后一个数据交换
  3. 将数组的长度减一的数组再次建堆，再次交换数据，
  4. 循环，直至最后
