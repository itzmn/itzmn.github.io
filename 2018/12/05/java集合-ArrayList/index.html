java集合-ArrayList

Posted on 2018-12-05 | In java
分析ArrayList源码实现，使用场景，自己实现一个ArrayList

简介

ArrayList是实现了List接口的集合，也是很常用的集合，是一个有序可重复的集合，是使用数组实现的

addAll(Collection c)方法，将一个集合添加进来

  1. 首先将传入的集合转成数组，并且计算长度
  2. 返回需要的数组（校验数组长度够不够，是否扩容）
  3. 对数组进行拷贝，将传入的数据拷贝到真正的数组
  4. 增加集合的size

addAll(int i,Collection c) 在某个索引处，添加一个集合

  2. 返回对应位置的数据

  1. 删除对应索引处数据

总结

  1. 集合内部的数组默认长度为10， 扩容长度为原数组长度的一半
  2. 集合默认构造器，使用了默认的空数组，当在第一次添加数据的时候，会根据这个判断该如何扩容，如果判断是默认的构造器，则使用默认的数组长度，如果不是则对其处理
  3. 添加方法，首先校验索引，再者判断现有的数组长度是否是需要的数组 长度，如果不够，则进行扩容
  4. 在进行数组拷贝的时候，需要对某个位置之后的数据进行拷贝，则使用System.copy()，
