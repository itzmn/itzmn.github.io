java集合-LinkedList

Posted on 2018-12-06 | In java
分析LinkedList源码实现，使用场景，自己实现一个LinkedList

简介

LinkedList是一个基于链表实现的集合，实现了List，Deque接口，具有多种性质，其内部的数据是有序的，可以按照顺序取值，并且因为其实现了队列接口，可以对其两端进行操作。

在队列越界的时候，操作方法一般分两类，一种是抛出异常的，一种是返回值的。具体在接口中定义，但是这个底层又是用链表实现的，所以添加元素不会越界，在得到或者删除元素的时候，会有区别

接口

其实现了List，Deque接口，我们看一下Deque接口，这个继承了Queue接口，首先我们先说一下队列，队列是先入先出的数据结构。尾部插入数据，头部移除数据

再看一下Deque接口，Deque是一个双端队列，意思是既可以从头添加数据也可移除数据，尾部也是一样,我们看一下它扩展的方法，这些方法有很多都是相同的。

总结

  1. 因为这个集合实现了Deque接口，不仅有list的方法，还有队列的操作，有很多方法都是对双端操作的。但是分两种，一种是在队列为空或者满的时候操作会抛异常，还有一种是不会，remove，add，get等大部分都会抛异常，poll offer peek 大部分都不会
  2. 因为这个底层是用链表实现的，所以插入比较方便，插叙效率就比较低了
