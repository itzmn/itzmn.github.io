总结

  1. 默认不可以存放key为null的节点，如果自定义了比较器可以添加

    因为默认是根据key转成比较器进行比较的，然后进行排序，null不可以，

  2. TreeMap默认根据key升序排列

  3. 迭代器是fail-fast的

  4. 是县城不安全的集合
