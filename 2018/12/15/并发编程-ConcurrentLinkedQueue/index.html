并发编程-ConcurrentLinkedQueue

发表于 2018-12-15 | 分类于 java ， 并发编程
字数统计: 1.4k 字 | 阅读时长 ≈ 5 分钟
ConcurrentLinkedQueue是一个支持并发的容器,是一个无阻塞算法实现的队列

简介

ConcurrentLinkedQueue是一个支持并发的容器，内部使用链表实现，是一个无界的安全队列，内部使用CAS实现。按照先入先出的顺序维护数据，新加入的数据放在尾部，移出的是头部的数据，不可以添加null数据。这是一个非阻塞的队列。

总结

  1. 内部链表添加删除，头指针和尾指针，并不一定是真正的头和尾，因为内部更新，两次才更新一次。
  2. 不可以添加null值
  3. 因为是无界队列，可能会添加很多值，当心内存溢出，并且尽量不要使用size，因为会遍历链表，浪费性能，查看是否为空，使用isEmpty
  4. 当队列为空，移除元素，也不会抛异常，返回null，
